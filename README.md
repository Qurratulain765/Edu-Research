# EduResearch Hub - Responsive Web Portal

A clean, modern, and fully responsive web landing page designed for an educational research organization. This portal showcases peer-reviewed insights, data-driven methodologies, and core research categories like Cognitive Science, EdTech, and Educational Policy.

This project was developed as a technical evaluation task for **Qwetrum Technologies**.

## 🚀 Features

* **Responsive Navigation:** A modern, sticky navbar that transitions smoothly into a mobile-friendly layout on smaller screens.
* **Hero Section:** A visually striking introduction with a clear Call-to-Action (CTA) directing users to core content.
* **Grid & Flexbox Layouts:** Utilizes contemporary CSS positioning techniques (Flexbox and Grid) to create beautifully structured informational cards.
* **Semantic HTML5:** Built using clean, search-engine-optimized, and accessible semantic elements (`<header>`, `<section>`, `<nav>`, `<footer>`).
* **Fluid Typography & Spacing:** Engineered using modern scale techniques for optimal readability across devices.

## 🛠️ Technologies Used

* **HTML5:** Structured layout and semantic web elements.
* **CSS3:** Custom styling, variables, layout management (Grid/Flexbox), transitions, and media queries for responsiveness.
